# Other Resources

This page contains a collection of GitHub repositories and external tools referenced in the MLCommons Science working group meeting notes.

## MLCommons Science & HPC Repositories
- [MLCommons Science](https://github.com/mlcommons/science) - Main working group repository.
- [MLCommons Science Benchmarks](https://github.com/mlcommons-science/benchmark) - Benchmark collection and source.
- [MLCommons HPC](https://github.com/mlcommons/hpc) - Reference implementations of MLPerf™ HPC training benchmarks.
- [MLCommons Policies](https://github.com/mlcommons/policies) - General policies for MLPerf™.
- [MLCommons Logging](https://github.com/mlcommons/logging) - MLPerf™ logging library.
